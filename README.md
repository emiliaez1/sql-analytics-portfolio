SQL QUERIES
This folder contains foundational SQL queries used in aanalysing data.


Topics Covered
- SELECT clause
- Sorting results with ORDER BY
- Limiting results with LIMIT
- Basic date filtering
- Aggregations and GROUP BY


Queries Explained
1. Daily Active Users
This query calculates how many unique users were active on each day.  
It uses `COUNT(DISTINCT user_id)` to avoid double-counting users.

2. Users per Country
This query shows how users are distributed across countries, which helps understand geographic reach.

3. Top Campaigns by Clicks
This query ranks marketing campaigns by total clicks to identify top-performing campaigns.
